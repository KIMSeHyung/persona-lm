# 범위

## 프로젝트 정의
`persona-lm`은 오픈소스 기반의 `digital mirror` 런타임이다.

사용자의 데이터로부터, 사용자를 완전히 복제하는 것이 아니라 "꽤 그 사람답게 느껴지는" 근사 모델을 만드는 것을 목표로 한다.

## 핵심 목표
다음 요소에서 사용자를 닮은 응답을 만들 수 있어야 한다.

- 말투와 설명 방식
- 취향과 선호
- 반복적으로 관심 보이는 주제
- 의사결정 경향
- 자기 서술 방식

## 비목표
- 완전한 인격 복제
- 모든 개인사에 대한 완벽한 사실 기억
- 초기 버전부터의 완전한 에이전트 자동화
- 처음부터 원격 SaaS 전제를 두는 구조
- 로컬 사용에 무거운 인프라를 요구하는 구조

## MVP 사용자 경험
사용자는 최소한 다음 흐름을 수행할 수 있어야 한다.

1. 개인 데이터를 import 한다.
2. 데이터를 구조화된 persona memory로 compile 한다.
3. persona runtime과 대화한다.
4. 왜 그런 답변이 나왔는지 inspection 할 수 있다.

## 성공 기준
- 응답이 사용자를 어느 정도 연상시킨다.
- 답변에 대한 근거 evidence를 추적할 수 있다.
- memory 시스템이 수정 가능하고 inspectable 하다.
- 로컬에서 최소한의 설정으로 실행된다.
